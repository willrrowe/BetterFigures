README - BetterFigures

Make figures for publication (in MATLAB)

Use the proPlot class to make a single plot. 
Use conFigure to combine many proPlot instances. This will display your plots in an array of pannels in a single figure.

See example.m for practical examples of how to use it.
See documentation of each class/function for more details.


To install:	
Download the files and add them to the MATLAB path.
